JAVA후기

7월을 마치면서...

2주간의 JAVA 학습이 끝났다.

SSAFY에 입과하기 전, 학교 수업 때문에 Python과 C는 다루어봤었다. 하지만 JAVA는 접해보지 못했기에 상당히 힘들 것이라 예상되었다. 월말평가도 끝나며 JAVA 수업이 끝났기에 약 1달간의 학습을 되돌아 보고자 한다.

프로그램 시작 전 ~ 스타트캠프

분반 테스트를 잘보기 위해서, 무엇보다도 앞으로 쓰게 될 언어인 만큼 미리 예습을 시작했다. 생활코딩 홈페이지에서 강의를 듣고 따라하면서 익숙해지고자 노력했다. 아무래도 전에 C를 배운적이 있어서 절차지향 부분까지는 상당히 익숙한 느낌이 들었지만, 아무래도 객체지향 부분의 경우 C에서는 다루지 않았고, Python에서도 필수적인 느낌은 아니었기에 익숙치 않았다. 답은 복습과 실습이라 생각해 강의에 나오는 코드는 모두 따라 쳐보고 강의를 듣고 즉시 강의 내용을 복습하며 공부했다.

본과정

짧은 스타트캠프도 끝나고 JAVA 비전공 분반에 들어가게 되어 본과정이 시작되었다. 단 2주 안에 프로그래밍 언어 하나를 마스터해야 했고, 내용의 깊이도 상당히 깊이까지 나가 어려웠다. 답은 복습이었다. 2시간의 라이브 강의를 들은 후에는 즉시 ppt와 코드를 보며 복습을 진행했고, 그날 나온 과제를 선택 과제까지 모두 수행하고자 했다. 하필이면 중간에 예비군 훈련이 잡혔었는데, 돌아오자마자 아메리카노를 들이키며 3일치를 주말 내내 공부했다. 덕분에 실력이 많이 늘었고, 첫 과목 평가도 괜찮은 성적으로 통과했다. 그렇게 본과정을 무사히 따라가고 7/28부로 본 과정이 끝나게 되었다.

앞으로의 다짐

원래 내 코테언어는 Python이었다. 하지만, 향후 JAVA를 많이 사용하게 될 것 같으니 JAVA를 깊게 파는게 좋을 것 같다고 생각해 주력 언어를 JAVA로 바꾸었다. 백준, SWEA 등에 있는 옛날에 풀었던 문제들을 JAVA로 다시 풀어보는 것부터 차근차근 하며 익숙해지고자 한다. 아무래도 시간이 지나면 많은 내용들을 까먹을 수 있으니 꾸준히 코테 문제를 풀다가 8, 9월 쯤에 JAVA의 정석 책으로 한번 더 복습할 예정이다. JAVA에 완전히 익숙해질 때까지 노력하겠다. - 2023.07.31
